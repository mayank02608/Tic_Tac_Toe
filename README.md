# 🎮 Tic-Tac-Toe Game in Python

A simple **console-based Tic-Tac-Toe game** built using Python. This project allows a player to play against the computer in the terminal. The computer makes random valid moves, making the game fun for beginners learning Python.

---

## 📌 Features

- ✅ Player vs Computer gameplay
- ✅ Simple text-based interface
- ✅ Random computer moves
- ✅ Win, Lose, and Draw detection
- ✅ Input validation
- ✅ Beginner-friendly Python code
- ✅ Uses functions for clean and modular programming
